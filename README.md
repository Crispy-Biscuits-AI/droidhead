# J0N1 (Johnny Number One) – Droidhead Assistant

This repository is the home for **J0N1**, the embodied AI assistant living inside the *Crispy Biscuits AI: Droidhead* project.

It is designed as a **template GitHub repository** you can clone and customize.

## Repo Overview

- `docs/` – Design docs, identity bible, state machine, getting started, etc.
- `prompts/` – System prompts for J0N1 and the orchestrator.
- `system/` – Knowledge files and state definitions used by your workflows.
- `workflows/` – Example n8n workflows or JSON exports.
- `assets/` – ASCII banners, logos, and visual references.
- `scripts/` – Helper scripts for deploying to Jetson / local stack.

## Quick Start

1. Clone this repo into your GitHub account.
2. Edit the files under `docs/` and `prompts/` with your preferred tooling.
3. Import any workflows under `workflows/` into n8n.
4. Mount `prompts/` and `system/` into your Jetson/n8n environment.
5. Start talking to J0N1.

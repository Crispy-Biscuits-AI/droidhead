# Changelog
All notable changes to **J0N1 (Johnny Number One)** and the **Crispy Biscuits AI: Droidhead Assistant** documentation are tracked here.

This project follows **Semantic Versioning (SemVer)**:
- **MAJOR** – Breaking persona/architecture changes
- **MINOR** – New features, behaviors, prompts, or submodules
- **PATCH** – Fixes, refinements, documentation clean-up

---
